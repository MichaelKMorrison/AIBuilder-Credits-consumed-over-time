# AIBuilder-Credits-consumed-over-time
This Power Automate Flow will pull historical records AI Builder credits used across all environments
Power Automate Flow: PullRecordsFrom_msdyn-AI EVENTS table in CDS Description: Used to connect to msdyn-aiEvents table in a tenant's Dataverse and pull all AI Builder credits consumed over time, with date of consumption. Flow pulls down entire tablespace so other relevant data can be used if needed
