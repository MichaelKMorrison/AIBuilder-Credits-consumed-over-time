# AIBuilder-Credits-consumed-over-time
This Power Automate Flow will pull each usage of AI Builder and quantity of credits used across all environments in a tenant. 

# Power Automate Flow: 
Pull Records From table msdyn-AIEVENTS in CDS 
# Description: 
Used to connect to msdyn-aiEvents table in a tenant's Dataverse and pull all AI Builder credits consumed over time, with date of consumption. Flow pulls down entire tablespace so other relevant data can be used if needed
